# הנחיות לעבודה על אתר משרד עורכי הדין לביא, ניסים-שלו.

## אודות הפרויקט

אתר תדמית למשרד עורכי דין המתמחה ברשלנות רפואית ונזקי גוף בישראל. קהל היעד: נפגעי רשלנות רפואית ובני משפחותיהם.

## מחסנית טכנולוגית

- HTML סטטי - קובץ בודד (index.html) ללא build step
- CSS טהור עם אנימציות keyframes (ללא framer-motion או ספריות חיצוניות)
- Google Fonts - Frank Ruhl Libre, Playfair Display, Assistant
- GitHub Pages - אירוח חינמי

## פקודות

אין build process. הקובץ index.html נפתח ישירות בדפדפן או דרך GitHub Pages.

להרצה מקומית: פשוט לפתוח את index.html בדפדפן.

## מבנה קבצים

```
/
├── index.html          # דף הבית (Hero + תחומי מומחיות)
├── style.css           # כל ה-CSS בקובץ נפרד
├── logo.svg            # סמל LNS
├── logo.png            # לוגו המשרד
├── README.md           # תיאור הפרויקט
└── CLAUDE.md           # קובץ זה
```

## עקרונות עיצוביים

- פלטת צבעים: רקע כחול כהה `#0F172A`, אקסנט ברונזה `#CD7F32`, לבן לטקסט
- אווירה: ארכיטקטונית, יוקרתית, רצינית - לא "AI generic", לא גרדיאנטים סגולים
- טיפוגרפיה: Frank Ruhl Libre / Playfair Display לכותרות (serif), Assistant לטקסט רץ
- כיוון: RTL (עברית)
- אלמנטים חוזרים: קווי ברונזה דקים, פינות מסוגרות, קו ידני מתחת לכותרות, pills מעוגלות

## קונבנציות קוד

- שם class-ים באנגלית קטנה עם מקפים (`credibility-card`, לא `CredibilityCard`)
- צבעים תמיד מהפלטה המוגדרת - לא להמציא hex חדשים
- טקסט עברי תמיד ב-`dir="rtl"`
- אנימציות תמיד דרך CSS keyframes, לא JS

## שיקולים משפטיים ואתיים

חשוב מאוד - עוסקים באתר של עורך דין:

- האתר חייב להכיל disclaimer: "פרסום זה אינו מהווה ייעוץ משפטי"
- אסור לכלול הבטחות תוצאה ("נבטיח לך פיצוי")
- נדרש ציות לכללי לשכת עורכי הדין בישראל לגבי פרסום
- דוגמאות תיקים מוצגות רק באופן כללי, ללא פרטים מזהים

## הערות לקלוד

- המשתמש מעדיף תשובות קצרות וממוקדות, step-by-step
- לא לבצע auto-correction לטקסטים בעברית - לשאול לפני שינוי
- בעת ציטוט ממסמכים שהועלו - לציין מקור במפורש
- להתייחס למשתמש כאל שותף מקצועי, לא כאל מתחיל
